# `docs/topics/` — Recursive Subtree

Drop new documentation files (and as many nested subdirectories as you
like) under this directory. Anything `*.md` here is automatically
discovered by `ailloy docs`, the TUI tree, and the `--docs` flag without
any code changes.

The top-level `docs/*.md` files remain the canonical entry points for
GitHub viewers; this subtree is for ancillary or grouped content
(tutorials, deep dives, per-team guides, etc.).
